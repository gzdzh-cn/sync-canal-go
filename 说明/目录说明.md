## 快速开始

### 1. 安装依赖

```bash
cd /Volumes/disk/site/go/dzhgo/dzhgo-admin

go get github.com/go-mysql-org/go-mysql
go get github.com/ClickHouse/clickhouse-go/v2
go get gopkg.in/yaml.v3
```

### 2. 创建同步程序

```bash
mkdir -p cmd/mysql_to_clickhouse
cp clickhouse部署/go-mysql同步方案/main.go cmd/mysql_to_clickhouse/
cp clickhouse部署/go-mysql同步方案/config.yaml cmd/mysql_to_clickhouse/
```

### 3. 修改配置

编辑 `cmd/mysql_to_clickhouse/config.yaml`:

```yaml
mysql:
  host: 127.0.0.1
  port: 13308
  user: sync_user
  password: sync_password_123

clickhouse:
  host: 127.0.0.1
  port: 9000
  database: dzh3136_go

sync:
  database: dzh3136_go
  tables:
    - clues
```

### 4. 运行同步

```bash
# 直接运行
go run cmd/mysql_to_clickhouse/main.go

# 或编译后运行
go build -o bin/mysql_to_clickhouse cmd/mysql_to_clickhouse/main.go
./bin/mysql_to_clickhouse
```

## 方案特点

### ✅ 优势

- **MySQL 8.4 支持** - 完全兼容 MySQL 8.x
- **ClickHouse 支持** - 直接写入 ClickHouse
- **指定表同步** - 可选择特定表同步
- **高性能** - 批量写入,低延迟
- **零依赖** - 纯 Go 实现,无需外部组件
- **完全控制** - 可自定义数据处理逻辑

### ⚠️ 注意事项

- 需要开启 MySQL Binlog (ROW 模式)
- 需要创建 MySQL 同步账号
- 首次部署需要先导入全量数据
- ClickHouse UPDATE/DELETE 性能较差

## 环境要求

### 必需

- ✅ Go 1.16+
- ✅ MySQL 8.4+ (开启 Binlog ROW 模式)
- ✅ ClickHouse 24.3+

### MySQL 配置

```ini
[mysqld]
log_bin = mysql-bin
binlog_format = ROW
binlog_row_image = FULL
server-id = 1
```
